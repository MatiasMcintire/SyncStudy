# Cambios de decoración — SyncStudy

Se agregó una capa visual nueva sin modificar la lógica principal del proyecto:

- `css/aesthetic.css`: rediseño aesthetic minimalista oscuro, glassmorphism, sombras, cards premium, responsive y animaciones.
- `js/aesthetic.js`: microinteracciones visuales seguras, glow según posición del mouse y feedback de presión.
- `index.html`: solo se añadieron las referencias a los dos archivos anteriores.

La lógica original de tareas, calendario, grupo, modal y storage se mantiene intacta.
